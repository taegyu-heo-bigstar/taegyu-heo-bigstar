## Projects / 프로젝트

### Local Processing Based Low-Cost Intelligent CCTV System

<details>
<summary>Click to expand!</summary>

A capstone design project that combines a Raspberry Pi-based CCTV device with a local desktop PC.  
The goal is to build a low-cost intelligent CCTV system that performs local video streaming and AI-based analysis without depending on cloud-based video processing. I was responsible for system design, chatbot implementation, and hardware setup in this project.

Raspberry Pi 기반 CCTV 장치와 로컬 데스크톱 PC를 결합하여, 클라우드 영상 분석에 의존하지 않는 저비용 지능형 CCTV 시스템을 개발하는 종합설계 프로젝트입니다. 저는 이 프로젝트에서 시스템 설계, 챗봇 구현, 하드웨어 구축 등을 담당하였습니다.

**Tech Stack / 기술 스택**

- Raspberry Pi
- Python
- OpenCV
- YOLO
- RTSP
- MediaMTX
- FastAPI
- PyQt5

**Main Features / 주요 기능**

- Real-time video capture using Raspberry Pi and a camera module
- RTSP-based video streaming to a local desktop PC
- Object detection and abnormal situation detection using computer vision and AI models
- Event notification through chatbot or external alert systems
- System design considering power failure, network instability, and local video backup

**Repository / 저장소**

- [Link](https://github.com/Eye-O-T/AI_CCTV)
